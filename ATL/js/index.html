This is implementation in progress of a NL-BATL*/BATL* tableau. It presently decides the NL-BATL* semantics but that is trivial to change.

Currently the only way to input a formula is to manually play with the URL.

The tableau determines the number of agents based on the largest agent# found in the formula. For example, f^-{2}f is satisfiable but f^-{1}f is not.

NOTE: this is a mostly complete protoype. Do not trust the output yet. Also this javascript version is even slower and less featureful than the native version
